# vuex-demo-cart

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### 功能点
- 产品列表页面
  - 数据渲染
  - 加入购物车
  - 数量合计，价格统计
- 购物车页面
  - 数据渲染
  - 全选非全选
  - 数量合计，价格统计
  - 修改商品数量
  - 删除商品
